1. **견적 받기** → 3D 파일 업로드 → 옵션 선택 → 견적 계산
2. **저장** → 견적 저장 (회원/비회원)
3. **장바구니 추가** → 원하는 견적을 장바구니에 담기

5. **주문하기** → 배송지 입력 → 주문 생성
6. **주문 완료** → 주문 번호 확인 → 다음 단계 안내
7. **마이페이지** → 주문 내역 확인
