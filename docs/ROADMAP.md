## 🎯 **즉시 시작 가능한 작업 (오늘)**

### 1️⃣ 주문하기 페이지 구현
```bash
# 파일 생성
app/checkout/page.tsx
```

**필요한 것**:
- 장바구니 데이터 가져오기 (`useCartStore`)
- 배송지 입력 폼 (이름, 전화번호, 주소, 우편번호)
- 주문 생성 API 호출
- 성공 시 `/order-complete?orderId=XXX`로 리다이렉트

### 2️⃣ 주문 완료 페이지 구현
```bash
# 파일 생성
app/order-complete/page.tsx
```

**필요한 것**:
- URL 파라미터에서 orderId 읽기
- 주문 상세 정보 표시
- 계속 쇼핑하기 버튼
