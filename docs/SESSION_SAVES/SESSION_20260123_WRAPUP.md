## ✅ 주요 달성 사항

### 1. 3D 뷰어 기능 고도화 (`components/canvas/Scene.tsx`)
기존 뷰어의 여러 문제점을 수정하고 고급 기능을 추가했습니다.
-   **초기 로딩 오류 해결**: `three-stdlib` 로더와의 호환성 문제(Buffer not defined)를 해결하기 위해 `fetch` 및 `arrayBuffer` 파싱 로직을 커스텀 구현하여 안정성을 확보했습니다.
-   **자동 피팅 완벽 구현**: 모델 로드 시 카메라가 너무 멀거나 가깝게 잡히는 문제를 해결하기 위해 **`useBounds` 훅**을 도입했습니다. 로딩 완료 0.1초 후 `bounds.refresh().clip().fit()`을 강제 호출하여 어떤 크기의 모델이든 화면 중앙에 완벽하게 맞춥니다.
-   **측정 도구 (Measurement Tool)**: "측정" 버튼을 토글식으로 변경하여, 필요할 때만 치수와 Bounding Box를 표시하도록 개선했습니다 (기본값: Off).
-   **기타 기능**:
    -   **로딩 스피너**: 대용량 모델 로딩 시 사용자 경험 개선.
    -   **OBJ 파일 지원**: STL 뿐만 아니라 OBJ 포맷 지원 추가.
    -   **색상 변경**: 8가지 프리셋 색상으로 모델 미리보기 커스터마이징.
    -   **스크린샷**: 현재 뷰를 PNG로 저장하는 기능.

### 2. 랜딩 페이지 리뉴얼 (`app/page.tsx`, `globals.css`)
단조로웠던 기존 디자인을 최신 트렌드에 맞춰 완전히 재설계했습니다.
-   **현대적 UI**: Glassmorphism(유리 질감), 그라디언트 텍스트, 부드러운 그림자 등을 적용하여 고급스러운 느낌을 주었습니다.
-   **동적 애니메이션**: `framer-motion`을 사용하여 스크롤 시 요소들이 부드럽게 나타나는 페이드인/슬라이드 효과를 적용했습니다. 배경에는 움직이는 Blob 애니메이션을 추가하여 생동감을 더했습니다.
-   **구조 개선**: Hero -> Features -> Process -> CTA 섹션으로 이어지는 논리적인 흐름을 구축했습니다.
